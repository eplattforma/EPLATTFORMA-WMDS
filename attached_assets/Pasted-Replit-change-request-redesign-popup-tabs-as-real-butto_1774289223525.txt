Problems with current result
the tabs extend beyond the popup width
a horizontal scrollbar appears
tabs are not clearly identified
they do not look like strong navigation controls
the UI feels cramped and unfinished
Required change

Redesign the top tab navigation of the customer offers popup so the tabs look like clear button-style navigation, fully visible inside the popup width.

This must apply to all 4 popup screens:

Offer Summary
Unused Offers
Offer-Driven Sales
All Active Offers
1) Replace the current scrollable tab strip completely
Remove
horizontal scrolling tab bar
scrollbar under the tabs
any overflow behavior where tabs go outside the popup width
weak text-only tab look

2) Make the tabs look like real buttons

Implement the tabs as button-like segmented navigation.

Visual direction

Each tab should look like a button, not just text with an underline.

Inactive tab/button
dark button background
visible border or subtle outline
strong readable label
clear padding
slightly rounded corners
Active tab/button
stronger filled background or accent background
brighter text
stronger border / glow / emphasis
clearly selected state at first glance
Hover state
brighter background or border on hover
pointer cursor
obvious interactivity
3) Fit all tabs inside the popup width

The 4 tabs must always stay within the popup width cleanly.

Preferred layout

Use one of these approaches:

Option A — 4 equal-width buttons in one row

If the popup width allows it:

4 equal-width buttons across the row
each button centered
text can wrap to 2 lines if needed
Option B — 2 x 2 button grid

If one-row layout is too tight:

place tabs in 2 rows of 2 buttons
this is preferred over any horizontal scrolling
must still look neat and intentional
Rule

Do not allow tabs to overflow horizontally.
If needed, wrap the labels inside the button rather than forcing one-line overflow.

center the text

5) Recommended structure

A good implementation would be:

container for tab buttons
CSS grid or flex layout

equal heights

rounded corners
