## Change 2 — `blueprints/cooler_picking.py` — Fix box_reopen redirect

        "cooler.box_reopened",

## Change 3 — `templates/cooler/route_picking.html` — Exclude cancelled boxes from overfull warning

Find this block near the top of `{% block content %}` (around line 36):

```jinja
  {# Check for overfull boxes (fill_pct > 100) using namespace to mutate inside loop #}
  {% set _overfull_ns = namespace(any=false) %}
  {% for _b in boxes %}{% if _b.fill_pct and _b.fill_pct > 100 %}{% set _overfull_ns.any = true %}{% endif %}{% endfor %}
  {% set _any_overfull = _overfull_ns.any %}
```

Replace with:
```jinja
  {# Check for overfull boxes — skip cancelled boxes #}
  {% set _overfull_ns = namespace(any=false) %}
  {% for _b in boxes %}{% if _b.fill_pct and _b.fill_pct > 100 and _b.status != 'cancelled' %}{% set _overfull_ns.any = true %}{% endif %}{% endfor %}
  {% set _any_overfull = _overfull_ns.any %}
```

What this fixes: the red "Box size too small" warning was still showing after the overfull box was cancelled and replaced with a correct box.
